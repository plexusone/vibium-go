## [v0.1.0] - 2026-01-31

### Highlights

- Go client for Vibium browser automation platform with full feature parity with JavaScript and Python clients

### Added

- Core types: `BoundingBox`, `ElementInfo`, `LaunchOptions`, `FindOptions`, `ActionOptions` ([`e6ba8d0`](https://github.com/grokify/vibium-go/commit/e6ba8d0))
- Custom error types: `ConnectionError`, `TimeoutError`, `ElementNotFoundError`, `BrowserCrashedError`, `BiDiError` ([`e6ba8d0`](https://github.com/grokify/vibium-go/commit/e6ba8d0))
- BiDi WebSocket client for browser communication ([`4f73d02`](https://github.com/grokify/vibium-go/commit/4f73d02))
- Clicker process management with binary discovery via `VIBIUM_CLICKER_PATH`, PATH, and cache directory ([`3f36cf0`](https://github.com/grokify/vibium-go/commit/3f36cf0))
- `Element` type with `Click()`, `Type()`, `Text()`, `GetAttribute()`, `BoundingBox()`, `Center()`, `WaitFor()` methods ([`b33b123`](https://github.com/grokify/vibium-go/commit/b33b123))
- `Vibe` browser control with `Launch()`, `Go()`, `Screenshot()`, `Find()`, `Evaluate()`, `Quit()` methods ([`b33b123`](https://github.com/grokify/vibium-go/commit/b33b123))
- Debug logging with `slog` via `VIBIUM_DEBUG` environment variable ([`48598c6`](https://github.com/grokify/vibium-go/commit/48598c6))
- Navigation methods: `Reload()`, `Back()`, `Forward()`, `WaitForNavigation()` ([`91ad5be`](https://github.com/grokify/vibium-go/commit/91ad5be))
- `FindAll()` method for finding multiple elements by CSS selector ([`91ad5be`](https://github.com/grokify/vibium-go/commit/91ad5be))
